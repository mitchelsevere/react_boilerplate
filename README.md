# react_boilerplate
A simple react boilerplate for starting projects

```bash
├── node_modules
├── public
│   ├── bundle.js
│   └── index.html
├── src
│   ├── components
│   │   └── [Home].js
│   ├── router
│   │   └── AppRouter.js
│   ├── scss
│   │   ├── base
│   │       ├── _base.scss
│   │       └── _vars.scss
│   │   ├── components
│   │   └── style.scss
│   └── app.js
├── .babelrc
├── package.json
├── README.md
└── webpack.config.js
```
